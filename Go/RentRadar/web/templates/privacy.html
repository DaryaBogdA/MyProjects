Политика конфиденциальности

Мы обрабатываем только данные, нужные для работы сервиса: контакты пользователя, объявления, сообщения и действия в личном кабинете.

Данные не передаются третьим лицам без законных оснований. Администратор может просматривать обращения, жалобы и модерацию для обеспечения безопасности.

Пользователь может запросить удаление аккаунта и связанных персональных данных через поддержку.
