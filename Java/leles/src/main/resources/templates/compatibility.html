Расчёт совместимости по дате рождения

Результат совместимости

Как рассчитывается:

Дата рождения одного партнера складывается с датой рождения другого и высчитывается одно большое число — сфера, по которой как раз-таки и определяется значение встречи двух людей.

Пример: Ваша цифра 10, партнёра 12. Складываем числа между собой и получаем 22.

Если вышло число больше 22, его нужно разложить на цифры и сложить их между собой. Например, 43 → 4+3 = 7.
